# Proyecto Calculadora (Trabajo Individual)

Este proyecto es una calculadora simple de consola desarrollada en Python.

El objetivo principal es demostrar el flujo de trabajo ágil simulando un entorno de equipo, utilizando:
* **GitHub** para el control de versiones.
* **Git-Flow** (uso de ramas y Pull Requests).
* **Trello (Kanban)** para la gestión de tareas.
